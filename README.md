# Franklin AI Middleware

Franklin is a privacy-first, local-aware AI assistant. It acts as a middleware layer that combines local tools with OpenAIâ€™s GPT-4o or TinyLlama fallback for hybrid intelligence.

## Features
- Read files and logs
- Summarize folders and notes
- Respond in natural language
- Local fallback with TinyLlama via Ollama
- Web UI + FastAPI backend
- Built-in test suite

## Quickstart

```bash
git clone https://github.com/warhorse407/franklin.git
cd franklin
docker compose -f docker-compose/docker-compose.yml up
```
